# Financial-market-dashboard
Interactive stock market dashboard built with Python, Dash and Plotly

##Features 
- Data cleaning and processing using Pandas
- Indexed price calculation for performance companison
- Interactive filtering by asset
- Time-series visualisation of stock prices

##Tech Stack
- Python
- Pandas
- Plotly
- Dash
 
 ## Dataset 
 Global Financial Markets dataset (Kaggle) 

## How to run
1. Install dependences: pip install pandas dash plotly
2. Run the app: python main.py
